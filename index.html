How to Get Started

1. Fork Repository

Start by forking on GitHub.

2. Clone It

Use this command:

git clone https://github.com/YourUsername/config-repo.git
3. Add Config

Create config.one in apps/your-app/.

4. Edit Config

Fill in details (use generator):

name: "My App"
author: "You"
version: "1.0.0"
description: "My awesome app."
logo_url: "https://example.com/logo.png"
download_url: "https://example.com/app.zip"
takedown: false
5. Push Changes

Commit and push:

git add apps/your-app/config.one
git commit -m "Add My App"
git push origin main
6. You're Live!

App appears after 8 PM IST update.
